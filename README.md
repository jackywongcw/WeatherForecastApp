# WeatherForecastApp

Simple iOS app that retrieve data from [OpenWeather](https://openweathermap.org)

### Instruction
Please sign up via the website to get your own api key
Replace the variable "apiKey" in:
```
NameConstant.swift
```
and build the app, should work fine
